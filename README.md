# CashLens（キャッシュレンズ）

キャッシュフローで見抜く、未来の資産を描く

## 概要

フリーキャッシュフロー（FCF）を軸とした独自スコアリングで企業の長期的健全性を評価し、将来の資産形成をインタラクティブにシミュレーションできる投資分析アプリです。

## 主な機能

- CFスコア：5指標を統合した0〜100点のキャッシュフロー健全性スコア（A〜Eグレード）
- 投資リターンシミュレーター：悲観・中立・楽観の3シナリオで将来資産を試算
- 企業ランキング：CFスコア順に企業を比較・スクリーニング

## 技術スタック

- Python
- Streamlit
- Pandas
- Plotly

## セットアップ

pip install -r requirements.txt

streamlit run main.py
